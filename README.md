# Lao-Works Dispatch Console

Dispatcher React + TypeScript app (Vite) implementing MVP flows:
- Login
- Dispatch queue
- Job detail
- Candidate score breakdown
- Assign/override
- Timeline
- Messaging panel

## Run
1. `npm i` (or `pnpm i`)
2. `npm run dev`

Set `VITE_API_BASE_URL` if gateway is not `http://localhost:4000/v1`.
